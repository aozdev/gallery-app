# Gallery App

![Gallery Screenshot](screenshot.png)

A minimalist and open-source image gallery to showcase visual inspiration.

## Features
- Dark / Light Mode
- Clickable images to enlarge
- Responsive design
- Footer with Discord / GitHub links

## How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Add images via `script.js` using `addEmbed(url, description)`
