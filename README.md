# Gallery App
You can let me know what features you'd like to see added on Discord. We plan to add more features over time. You can also use this repo as a draft and make changes to its design by forking the project to incorporate the changes. Support our development by giving us a star.

## Features
- Dark / Light Mode
- Clickable images to enlarge
- Responsive design
- Footer with Discord / GitHub links

## How to Use
1. Clone the repository
2. Open `index.html` in your browser
3. Add images via `script.js` using `addEmbed(url, description)`
